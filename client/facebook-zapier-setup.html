Facebook Events Integration

Automatically sync your Facebook page events to MarketPace calendar

1

Benefits of Integration

  • Automatic event synchronization from Facebook to MarketPace
  • Cross-platform promotion reaches both Facebook and local audiences
  • Real-time updates when events are modified on Facebook
  • Preserve event details, images, and ticket links
  • Increase local event visibility and attendance
2

Connect Your Facebook Page

Find this in your Facebook Page settings under "Page Info"
Get this from your Zapier webhook trigger
4

Create Zapier Automation

Set up the automation workflow in Zapier:

Trigger: Facebook Pages → New Event Action: Webhooks → POST Request Webhook URL: https://workspace.ihavecreativeid.repl.co/api/zapier/facebook-events/webhook Payload (JSON): { "event": { "id": "{{event_id}}", "name": "{{event_name}}", "description": "{{event_description}}", "start_time": "{{start_time}}", "end_time": "{{end_time}}", "location": "{{location}}", "cover": {"source": "{{cover_photo}}"}, "ticket_uri": "{{ticket_link}}" }, "user_id": "YOUR_MARKETPACE_USER_ID", "facebook_page_id": "{{page_id}}", "zapier_signature": "{{zap_meta_human_now}}" }
Create Zapier Automation
5

Test Integration

After setting up your Zapier automation:

  • Create a test event on your Facebook page
  • Wait 2-3 minutes for Zapier to process
  • Check your MarketPace calendar for the imported event
  • Verify event details, location, and links are correct
⚠️ Note: It may take up to 15 minutes for new events to appear in Zapier's trigger list.
6

Troubleshooting

Common Issues:

  • Events not syncing: Check Zapier task history for errors
  • Missing event details: Ensure Facebook event has complete information
  • Webhook errors: Verify the webhook URL is correct and accessible
  • Facebook permissions: Ensure you have admin access to the Facebook page

Need help? Contact MarketPace support with your Zapier task ID and error details.
