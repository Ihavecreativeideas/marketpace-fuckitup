## HOW TO UPDATE:
1. Click on the file name
2. Click the pencil icon (✏️ Edit) 
3. Delete all content (Ctrl+A, Delete)
4. Copy the updated content from Replit
5. Paste into GitHub
6. Click "Commit changes"
